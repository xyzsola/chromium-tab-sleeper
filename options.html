Waktu tidur

Tab akan ditidurkan (discard) setelah tidak aktif selama durasi ini. Set 5 detik untuk langsung menidurkan tab begitu Anda pindah tab.

Catatan: Chrome membatasi ketepatan waktu latar belakang. Untuk durasi di bawah ±30 detik, tab tetap akan tertidur, namun waktunya bisa meleset beberapa detik saat browser sedang idle.

Pengecualian situs

Tab dari situs berikut tidak akan pernah ditidurkan. Gunakan nama domain (contoh: youtube.com) atau wildcard subdomain (contoh: *.google.com).
